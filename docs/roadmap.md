# Roadmap

## Near-Term

- DOT / Mermaid graph export for review materials
- Better `graph` output formats such as Mermaid or DOT
- Optional stricter mode for non-phony tasks without outputs
- Command timing summaries and lightweight benchmark reporting

## Next Phase

- Dynamic ready-queue scheduler instead of level batches
- Remote or shared cache backends
- Plugin hooks for common transforms
- Built-in helpers for Markdown, copy, and archive tasks

## Stretch Goals

- Persistent file metadata cache for large trees
- Sandboxed execution adapters
- Optional worker protocol for distributed execution
